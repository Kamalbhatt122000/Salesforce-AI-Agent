Create a Connected App

Follow these steps in your Salesforce org to enable OAuth access

1

Open Setup

Log into your Salesforce org → click the ⚙️ Gear icon (top-right) → select Setup

2

Navigate to App Manager

In the Quick Find box, type App Manager → click App Manager

3

Create New Connected App

Click New Connected App (top-right button)

4

Basic Information

Connected App NameSalesforce AI Agent
API Name(auto-fills)
Contact Emailyour-email@example.com
5

Enable OAuth Settings

Check ✅ Enable OAuth Settings, then configure:

Callback URL http://localhost:5000/callback
Selected OAuth Scopes Full access (full)
Perform requests at any time (refresh_token, offline_access)
6

Save & Copy Credentials

Click Save → then Continue. On the next page, click Manage Consumer Details. You'll get:

Consumer KeyCopy this → paste in Step 2
Consumer SecretCopy this → paste in Step 2

You're Ready!

Once you have the Consumer Key and Consumer Secret, click the button below to proceed.
